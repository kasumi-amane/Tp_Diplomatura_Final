
Acceso a BD: https://cloud.mongodb.com/

El proyecto esta en proceso, esta construido en Node para el back y Hbs para el front.

La idea del proyecto es hay una home a la que se accede solo iniciando sesion. 
En la home Podes cargar posteos para cada dia de la semana y si queres tambien podes eliminarlos. (No pude generar una sesion para cada usuario individual)

Se aplica el crud en la creacion, edicion y eliminacion de usuarios. 

Se utiliza un autenticador para que usuarios no registrados no puedan entrar a todas las rutas.
